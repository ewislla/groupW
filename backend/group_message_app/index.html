Group Message App API Documentation

Base URL: http://localhost:8000/api

Note: All endpoints currently expect data to be sent as application/x-www-form-urlencoded (Standard Form Data).

POST /RegisterApi.php

Creates a new user account, hashes their password, and returns a secure API authentication token.

Request Parameters

Parameter Type Required Description
name String Yes The user's display name. Must be unique. Minimum 3 characters.
password String Yes The user's password. Minimum 6 characters.

Responses

Success (200 OK)

{
    "status": "success",
    "message": "User registered successfully",
    "api_token": "4a9b1c8f3e2d7a6b5c4d3e2f1a0b9c8d7e6f5a4b3c2d1e0f9a8b7c6d5e4f3a2b"
}

Error (Validation or Conflict)

{
    "status": "error",
    "message": "Error: Username is already taken."
}
POST /LoginApi.php

Authenticates an existing user and retrieves their active API token for subsequent requests.

Request Parameters

Parameter Type Required Description
name String Yes The registered username.
password String Yes The associated password.

Responses

Success (200 OK)

{
    "status": "success",
    "message": "Login successful",
    "api_token": "4a9b1c8f3e2d7a6b5c4d3e2f1a0b9c8d7e6f5a4b3c2d1e0f9a8b7c6d5e4f3a2b"
}

Error (Invalid Credentials)

{
    "status": "error",
    "message": "Incorrect password"
}
